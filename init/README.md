

Objectives

- have capabilities to filtered a syslog flow and forward them to another syslog server
- could be run as micro-services
- provide error management
- simulator capabilities
- performance counter
- cli to configure
- yaml to configure
- webui to review status
- dockerized version
- small footprint
- SBOM
- tls configuration
- other tcp 
- could support RFC 3164 (legacy) and RFC 5424 (modern) message format.

Syslog RFC
- RFC 6012 – Datagram Transport Layer Security (DTLS) for Syslog
- RFC 6587 – Transmission of Syslog Messages over TCP
- RFC 5426 – Syslog over UDP
